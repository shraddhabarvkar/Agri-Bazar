# Agri-Bazar
Agri Bazaar is a full-stack web application that provides a digital marketplace connecting farmers directly with buyers. The platform enables farmers to list agricultural products, manage inventory, and process orders, while buyers can browse products, place orders, and track their purchases.

The goal of Agri Bazaar is to simplify agricultural trading by eliminating intermediaries, improving market accessibility, and providing a secure and user-friendly shopping experience.
